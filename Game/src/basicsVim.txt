:set number - turns on line numbering
i - insert mode
ESC - command mode
:q - quit Vim
:w - write/save state of file
:wq - save and quit
:q! - quit without saving
dd - delete the current line
#dd - delete # number of lines (including the one we're currently at)
u - undo changes
ctrl+r - redo changes
/yourText+ENTER - search for a string
:%s/yourText/replaceText/g(+c) - find and replace text

These are the basic Vim commands!
